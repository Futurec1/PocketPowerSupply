## Project Overview

The Pocket Power Supply is a versitile, multi-rail, mobile DC power supply. It can accept a wide range of DC power sources (USBC PD/PPS/non-PD, internal battery, external jack), and output a fixed 3.3V, a fixed 5V, and 2 adjustable output rails. it is meant to be a convienient prototyping tool, makeshift power supply, and a low-power UPS. A jack of all trades, master of none, but hella conveneint.

## Educational Value Added

Advanced soldering, Power management (safety, battery charging, power ORing, DCDC conversion, varying power input and outputs), Embedded Systems (making IC's interact with each other), thermal management
